How to setup an account

  1. Move the mouse cursor to "Account" in the top right corner so that the dropdown appears.


  2. Click on "Register" in the list.


  3. Enter your personal details in each text box as labelled.

    The following information is required and are labelled on the page with an asterisk (*):

    • Title
    • First Name
    • Last Name
    • Email
    • Password
    • Confirm Password
    • Address Line 1
    • Town/City
    • Postcode

      • It is recommended that you select to be contacted "By Email" or "By Text" so that you will be notified if the status of a stolen bicycle of yours changes.


  4. Click "Create Account" at the bottom of the page.
